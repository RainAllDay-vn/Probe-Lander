### Installing Unity
Download and install Unity 2022.3 via Unity Hub: https://unity3d.com/get-unity/download

### Installing Anaconda

Go to https://www.anaconda.com/download and install anaconda

### Installing ML-Agents

1. Set Up Python Environment (with Conda)
conda create -n mlagents python=3.10.12 && conda activate mlagents

2. Clone ML-Agents Repository
git clone --branch release_22 https://github.com/Unity-Technologies/ml-agents.git
cd ml-agents

3. Install ML-Agents Python Packages
python -m pip install ./ml-agents-envs
python -m pip install ./ml-agents

4. Install PyTorch with CUDA (Windows only)
pip3 install torch~=2.2.1 --index-url https://download.pytorch.org/whl/cu121

5. Add Unity Packages
Open Unity > Window > Package Manager > Unity Registry
Search for ML Agents and install

### Set Up Environment
The environment is already configured in the Smooth Surface Unity file.

###Training with Genetic Algorithm
- Open the project in unity 2022.3
- Press Play to start training

###Training with Deep Q-Learning
- Use the Deep Q Unity build
- Train via Unity Environment Gym Wrapper

###Training with PPO
- Go to the PPO build folder
- Train with:
mlagents-learn config.yaml --env=<path_to_environment>
